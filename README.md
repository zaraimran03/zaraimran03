## Featured Project

### CulinaryAI Pro
> AI-powered recipe web application · React · Django · Django REST Framework · SQLite · Google Gemini API

An intelligent culinary assistant that generates personalised recipes using the Gemini API. Built with a React frontend and Django backend, it features a **Generational Adaptation Engine** that tailors recipes to dietary needs and skill levels, and an **Antigravity Mode** (Inverse Culinary Intelligence) for experimental recipe generation.

`React` `Django` `DRF` `Gemini API` `SQLite` `Python` `JavaScript`
